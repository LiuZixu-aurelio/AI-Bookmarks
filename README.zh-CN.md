# AI 书签

[English](./README.md) | **中文**

一个精心整理的 AI 工具、研究资源与开发实用链接集合。

本仓库收集高质量的 AI 相关链接，目标是持续更新的 AI 知识导航目录。

---

## 浏览器兼容性

`bookmarks.html` 采用 Netscape Bookmark File Format 标准，大多数现代浏览器都支持导入。

支持的浏览器包括（但不限于）：
- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Safari
- Brave
- Opera

---

## 如何使用

### 浏览

你可以直接打开 `bookmarks` 文件夹中的 Markdown 文件进行查看。

示例分类：
- AI Tools
- AI Models
- Learning Resources
- Research Papers
- Datasets
